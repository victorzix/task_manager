### Requirements
Before running, you will need to install [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/)

## Installation

### Clone this repository

```
git clone "https://github.com/victorzix/task_manager"
cd [FOLDER NAME]
```

### Install dependencies

```
npm install
```

### Run the application

```
npm run start:dev
```
